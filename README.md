# Lets Emoji

#### 介绍
这是一款可以用中文搜索Emoji关键字的WPF APP。
可以联网更新搜索词库，虽说一般来说不需要联网更新。

#### 软件架构
C# + WPF

#### 使用说明
Win7、8、8.1用户请先下载字体Segoe UI Emoji：
http://funjoo.fun/download/seguiemj.ttf
